# Трекер прочитанных книг

Консольное приложение для учёта прочитанных книг с модульной архитектурой.

## Запуск

```bash
python main.py
```

## Меню

```
1. Добавить книгу
2. Показать все книги
3. Показать среднюю оценку
4. Статистика по авторам
5. Удалить книгу
6. Выход
```

## Структура модулей

| Модуль | Назначение |
|---|---|
| `main.py` | Точка входа: меню, ввод-вывод, вызов функций из других модулей |
| `storage.py` | Чтение и запись JSON, поиск, удаление |
| `stats.py` | Подсчёт средней оценки, статистика по авторам |
| `models.py` | Класс `Book` (поля: автор, название, оценка, дата) |

## Поля книги

- **Название** — строка
- **Автор** — строка
- **Оценка** — целое число от 1 до 5
- **Дата прочтения** — строка в формате ГГГГ-ММ-ДД

Данные хранятся в файле `books.json`. При добавлении книга проверяется на дубликат (автор + название).
